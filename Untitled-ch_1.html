Problem Statement

Title: Tic Tac Toe Game

Description:
You are required to build a functional Tic Tac Toe game using HTML, CSS, and JavaScript where two players can alternately take turns to mark "X" and "O" on a 3x3 grid. The game should detect a win, draw, or ongoing play in real-time, highlighting the winning combination when a player wins and preventing further moves after the game ends. An optional "Player vs Computer" mode can be included.

Functional Requirements:

  • The grid must be 3x3 with 9 clickable cells.
  • Players should alternate turns, starting with "X".
  • The game must immediately detect and declare a winner or a draw.
  • Once the game is over (win or draw), no more moves should be allowed until the game is reset.
  • A "Reset" or "New Game" button must be provided to start a new game.
